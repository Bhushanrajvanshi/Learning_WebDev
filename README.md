How To Setup TypeScript in Project Step-by-Step

Step: 1 -> 

initialize the project

  -> npm init -y

  -> npm install -D typescript


Step: 2 -> 

tsconfig.json File Initialization

-> npx tsc init


Step: 3 -> 

How to compile ts file

-> npx tsc


Step: 4 ->

How to run js file

-> node dist/filename.js
